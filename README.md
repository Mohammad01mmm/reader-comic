# Reader Comic 📖🎨

A simple web-based comic book reader built with **HTML, CSS, JavaScript, and jQuery**.

## Features 🚀
- 📚 Easy navigation between pages  
- 🔍 Zoom in/out for better reading  
- ⏩ Keyboard shortcuts for page navigation  
- 🌙 Dark mode support
